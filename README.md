# 🧠 Local Knowledge Assistant (RAG-based)

This project is a **Retrieval-Augmented Generation (RAG)** based AI assistant built using **LangChain and Ollama**.

## 🚀 Features

* Runs LLM locally using Ollama
* Processes custom data (text files)
* Uses embeddings and vector database (FAISS)
* Retrieves relevant context
* Answers strictly from provided data
* Reduces hallucination using prompt engineering

## 🛠️ Tech Stack

* Python
* LangChain
* Ollama
* FAISS

## ⚙️ How It Works

1. Load data
2. Split into chunks
3. Convert to embeddings
4. Store in vector DB
5. Retrieve relevant context
6. Generate answer

## ▶️ How to Run

```bash
pip install -r requirements.txt
ollama run llama3
python app.py
```
